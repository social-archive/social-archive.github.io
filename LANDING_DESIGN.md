# 랜딩 페이지 디자인 - 멀티프로필

## 컨셉

**멀티프로필**이 랜딩 페이지의 핵심입니다.

- 랜딩 페이지 = 멀티프로필 허브
- 여러 프로필(무료자료실, 블로그, 멤버십스토어 등)을 한 화면에서 선택

## 구조

```
src/pages/index.js
├── profiles[]     ← 프로필 목록 (여기에 추가하면 카드 자동 생성)
├── ProfileCard    ← 개별 프로필 카드 컴포넌트
└── hero + profiles 섹션
```

## 현재 프로필

- 유튜브, 인스타그램, 카카오톡 (외부 링크 - URL 수정 필요)
- 무료자료실, 블로그 (내부)
- 멤버십스토어 (외부)

## 프로필 추가 방법

`index.js`의 `profiles` 배열에 객체 추가:

```js
{
  id: 'unique-id',
  title: '프로필명',
  description: '설명',
  to: '/내부경로',        // 내부 링크
  // 또는
  href: 'https://...',    // 외부 링크
  external: true,
  icon: '📦',
}
```

## 확장 시 고려사항

- 프로필이 4개 이상이면 그리드가 `auto-fit`으로 자동 배치
- 프로필별 이미지/썸네일 추가 시 `icon` 대신 `image` prop 사용 가능
- 프로필별 색상/테마는 `profile.theme` 등으로 확장 가능
