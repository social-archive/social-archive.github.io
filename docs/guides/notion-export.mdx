# Notion Markdown 내보내기

랜딩 페이지 콘텐츠는 **Notion에서 Markdown으로 내보낸 후** `src/pages/_home.mdx`에 반영하는 방식으로 관리합니다.

## 워크플로우

1. **Notion에서 내보내기**
   - 해당 페이지 우측 상단 `⋯` → **Export** → **Markdown & CSV** 선택
   - ZIP 다운로드 후 압축 해제

2. **콘텐츠 반영**
   - `src/pages/_home.mdx` 파일을 열고
   - 내보낸 `.md` 파일 내용을 복사해 붙여넣기
   - 필요 시 MDX 문법(컴포넌트, import)으로 보완

3. **빌드**
   - `npm run build` 또는 `npm run start`로 확인

## 참고

- Notion 내보내기 Markdown은 기본 마크다운 문법을 따릅니다.
- 이미지는 `static/img/` 등에 넣고 경로를 수정해야 할 수 있습니다.
- `@site/src/components/Notion`의 Callout, Card 등 컴포넌트를 함께 사용할 수 있습니다.
