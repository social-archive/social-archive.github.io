# 소셜아카이브 포털

[Docusaurus](https://docusaurus.io/)로 구축한 소셜아카이브 포털입니다.

## 설치

```bash
npm install
```

## 로컬 개발

```bash
npm start
```

개발 서버가 실행되며, 대부분의 변경사항이 실시간으로 반영됩니다.

## 빌드

```bash
npm run build
```

`build` 폴더에 정적 파일이 생성됩니다.

## 배포

GitHub Actions로 `main` 브랜치 푸시 시 자동 배포됩니다. (`.github/workflows/deploy.yml`)
